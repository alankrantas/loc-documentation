# LOC Documentation (Archived)

**LOC Documentation** (or LOC-Doc) is the official user guide and reference for [**FST Network**](https://www.fst.network/)'s cloud-native data product platform, [**Logic Operating Centre (LOC)**](https://www.fst.network/logic-operation-centre).

The site is built with [**Docusaurus 3.0**](https://docusaurus.io/) and deployed to AWS ECR.

## Docs Index

This site currently contains the following docs:

### General Docs

-   [LOC-Docs Revision](/docs/revision)
-   [System FAQs](/docs/category/system-faqs)
-   [About and Contact](/docs/about)
