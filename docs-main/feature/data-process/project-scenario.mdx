# Manage Project and Scenario

To create projects and scenarios. A data process belongs to a scenario, which in turn belongs to a project.

An API route trigger also have to be in a project.

## Edit or Delete a Project or Scenario

<Tabs>
  <TabItem value="studio2" label="LOC Studio" default>

Select a project or scenario and click **Edit \<item\>** <img src={require("./img/Edit.png").default} width="32px" alt="Save" className="img-bg" /> or **Delete \<item\>** <img src={require("./img/Delete.png").default} width="32px" alt="Delete" className="img-bg" /> on the top right.

  </TabItem>
</Tabs>

:::warning
If a data process is linked to one or more triggers, deleting the project or scenario containing it will make the triggers inoperable.
:::
