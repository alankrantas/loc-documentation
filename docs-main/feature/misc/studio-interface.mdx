# Studio Interface

Available interface options in LOC Studio.

## Signing In

:::info Creating an Account
Please contact the user with administration privilege to create a new account for you with desired username and email.

LOC Studio may prompt you to change the temporary password (if one is set) after signing in the first time.
:::

1. Open the URL of your LOC Studio instance in the web browser.

<div className="center-padded-sm">
    <img src={require("./img/Welcome.png").default} alt="Welcome" />
</div>

2. Click **Start**, enter email/username and password then click **Sign in**.

<div className="center-padded-sm">
    <img src={require("./img/Login.png").default} alt="Login" />
</div>
