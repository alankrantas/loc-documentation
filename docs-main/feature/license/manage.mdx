# View and Manage License

To inspect the license and update it if necessary.

## Inspect License

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs>

<div className="center-padded-sm">

<div className="center-padded-sm">

</div>

## Update License

<Tabs>
  <TabItem value="studio2" label="LOC Studio" default>

1. Click **Update License** on the top right.

<div className="center-padded-sm">
    <img
        src={require("./img/Update-license.png").default}
        alt="Update-license"
    />
</div>

2. Select a license file and preview the content, then click **Apply** to upload.

<div className="center-padded-sm">
    <img
        src={require("./img/License-preview.png").default}
        alt="License-preview"
    />
</div>

3. Wait for the activation to complete.

<div className="center-padded-sm">
    <img
        src={require("./img/License-activation-successful.png").default}
        alt="License-activation-successful"
    />
</div>

  </TabItem>
</Tabs>
