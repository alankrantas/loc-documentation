# QueryString Parser

Parse QueryString from the HTTP request into an object.

| Logic type | Input           | Output                                                                       |
| ---------- | --------------- | ---------------------------------------------------------------------------- |
| Generic    | `ctx.payload()` | Session variable `params` (type: `any` or `null` if not parsed successfully) |

If the QueryString is `?name=Arthur-Dent&age=42`, the `params` object will be parsed into

```javascript

    name: "Arthur-Dent",
    age: 42
}
```
