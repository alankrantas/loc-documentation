# Payload JSON Parser

Parse trigger payload to JSON and write to the session storage.

| Logic type | Input           | Output                                  |
| ---------- | --------------- | --------------------------------------- |
| Generic    | `ctx.payload()` | Session variable `parsed` (type: `any`) |

:::note
This logic assumes payload is not empty. `JSON.parse` will also throw an error if the decoded payload is not validated JSON.
:::
